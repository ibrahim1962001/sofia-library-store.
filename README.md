🛍️ Sofia Library Store | متجر مكتبة صوفي
متجر إلكتروني متكامل مصمم للمشاريع الصغيرة، يتميز بلوحة تحكم ذكية ونظام إدارة مخزون يعتمد على تقنيات Google.

🚀 المميزات الرئيسية (Key Features)
لوحة تحكم كاملة (Admin Panel): لإدارة المنتجات، البوكسات، ومتابعة الطلبات والإحصائيات.

نظام سلة المشتريات (Cart System): تجربة تسوق سلسة مع إمكانية إتمام الطلب عبر WhatsApp.

قاعدة بيانات سحابية (Cloud Database): استخدام Google Sheets كقاعدة بيانات عن طريق Google Apps Script API.

إدارة المحتوى: إمكانية إضافة وتعديل وحذف المنتجات والعروض مباشرة من لوحة التحكم.

نظام آراء العملاء: تجميع تقنيات ومراجعات العملاء مع نظام موافقة الإدارة.

🛠️ التقنيات المستخدمة (Tech Stack)
Frontend: HTML5, CSS3, JavaScript (ES6 Modules).

Backend: Google Apps Script (Serverless API).

Database: Google Spreadsheets.

Media Hosting: ImgBB API / Google Drive.
